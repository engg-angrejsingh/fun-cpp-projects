# 🚀 Fun C++ Projects

A collection of **beginner-friendly C++ projects** built to practice programming concepts and improve problem-solving skills.

Each project is a standalone terminal application focusing on a specific concept, making it easy to learn, explore, and contribute.

---

## 📂 Projects

| No. | Project | Description |
|:---:|---------|-------------|
| 01 | [🎨 Color Guessing Game](./01-color-guessing-game) | Guess the randomly selected color in a fun terminal-based game. |
| 02 | [🧮 Scientific Calculator](./02-scientific-calculator) | A colorful terminal-based scientific calculator with arithmetic and scientific operations. |

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/engg-angrejsingh/fun-cpp-projects.git
```

### Navigate to the Repository

```bash
cd fun-cpp-projects
```

### Open a Project

Example:

```bash
cd 01-color-guessing-game
```

or

```bash
cd 02-scientific-calculator
```

Compile the project:

```bash
g++ main.cpp -o app
```

Run the executable:

**Windows**

```bash
app.exe
```

**Linux / macOS**

```bash
./app
```
