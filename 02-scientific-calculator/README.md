# 🧮 Scientific Calculator (C++)

A terminal-based **Scientific Calculator** built in **C++** with a colorful console interface. It supports both basic arithmetic and scientific operations while providing input validation, previous result memory, and a clean menu-driven experience.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- % Modulus
- 🔲 Square
- 🧊 Cube
- ⚡ Power
- 📐 Square Root
- 📊 Logarithm
- 🌊 Sine (`sin`)
- 🌊 Cosine (`cos`)
- 🌊 Tangent (`tan`)
- 💾 Continue calculations using the previous result
- 🎨 Colored terminal interface
- ✅ Input validation
- ❌ Division by zero handling
- 🖥️ Cross-platform terminal clear (Windows, Linux, macOS)

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/engg-angrejsingh/fun-cpp-projects.git
```

### Compile

**Windows (g++)**

```bash
g++ main.cpp -o calculator.exe
```

**Linux / macOS**

```bash
g++ main.cpp -o calculator
```

### Run

**Windows**

```bash
calculator.exe
```

**Linux / macOS**

```bash
./calculator
```
